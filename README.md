# PIXEL CONTROL PROJECT

This project is a conbination of the ManiaControl `Pixel Control Plugin` and the `Pixel Control Server` API. The goal of this project is to have a modular way to control many ManiaPlanet servers via ManiaControl and the `Pixel Control Plugin` from the `Pixel Control Server` to assign matches, get stats from matches and players, etc...

The main goal is to be modular : the `Pixel Control Server` is a Docker Container that deploys an API that will be linkable to many ManiaPlanet+maniControl server with `Pixel Control Plugins`, and from that do everything you want with the exposed API of the `Pixel Control Server`, create a MatchMaking website with custom teams, and ELO ranking, a tournament conpetitve website, a practice server to find team mates and other teams to have serious matches, ...
