# Maniaplanet server

This a dockerized version of a maniaplanet, Shootmania server.

# Mandatory environement variables

`DATABASE_NAME`: Maniacontrol MYSQL database name

`XMLRPC_PORT`: Maniacontrol XMLRPC port (should start at 5000)

`MYSQL_USERNAME`: Maniacontrol MYSQL database username

`MYSQL_PASSWORD`: Maniacontrol MYSQL database password

`MYSQL_ADDRESS`: Maniacontrol MYSQL database host address

`MYSQL_PORT`: Maniacontrol MYSQL database port

`SERVER_NAME`: Server name without spaces

`DEDICATED_USERNAME`: Server dedicated account username

`DEDICATED_PASSWORD`: Server dedicated account password

# Optionnal environement variables

`TITLE_PACK`: Shootmania titlepack game mode name (optionnal, defaults to: SMStormElite@nadeolabs)
